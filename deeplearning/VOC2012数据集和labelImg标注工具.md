# labelImg标注工具
官方地址：https://github.com/tzutalin/labelImg

## 安装(Ubuntu)

```bash
pip install labelImg
```
使用
```bash
labelImg [IMAGE_PATH] [PRE-DEFINED CLASS FILE]
```
